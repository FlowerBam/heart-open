# 캐릭터 이미지 [TODO_OPERATOR]

매칭 슬롯머신 결과에 사용되는 캐릭터 이미지를 이 폴더에 업로드해주세요.

## 파일

- `e.png` — Emotional (감정 표현형)
- `s.png` — Social (함께하기형)
- `a.png` — Adaptive (유연한 적응형)
- `r.png` — Reliable (신뢰 기반형)
- `p.png` — Practical (실용 즐거움형)

## 규격

- 정사각형 (권장: 1024×1024 또는 512×512)
- 배경: 투명 PNG
- 5개 모두 일관된 톤·스타일 (운영자 책임)

이미지가 아직 없으면 슬롯/결과 카드는 자동으로 알파벳 placeholder(E/S/A/R/P)로 fallback 됩니다.
